# Client Scenario

## Organisation

The simulated client is a small UK-based SME with approximately 20 employees.

The company operates a small internal network supporting office operations and a basic web presence.

## Infrastructure

The internal network contains the following systems:

- Windows 10 workstation used by employees
- Ubuntu server hosting internal services
- Legacy Linux system running outdated services
- Web services accessible internally

## Objective

Tanasiom Aegis has been engaged to perform an internal security risk review.

The objective of the engagement is to identify exposed services, potential vulnerabilities, and security weaknesses within the internal infrastructure.

The assessment focuses on:

- service exposure
- outdated software
- insecure configurations
- potential attack paths

## Scope

In-scope systems:

- Ubuntu Server
- Windows 10 workstation
- Metasploitable legacy host

Testing will be conducted from an internal assessment workstation using Kali Linux.
