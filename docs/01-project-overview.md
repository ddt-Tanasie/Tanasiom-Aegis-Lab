# Project Overview

## Purpose
This repository contains the technical lab environment used to support the Tanasiom Aegis security assessment concept.

## Context
The lab simulates a small internal SME-style network in which controlled reconnaissance, service enumeration, vulnerability validation, and remediation planning can be demonstrated.

## Current Lab Components
- Kali Linux attacker workstation
- Windows 10 workstation
- Ubuntu Server target
- Metasploitable vulnerable target

## Objective
To provide a practical technical environment that supports assessment methodology, evidence gathering, and portfolio-grade cybersecurity demonstration.
